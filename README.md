## 🐱 소개

![UMO](./assets/character/umo-body.png)

UMO는 우산을 필요한 순간에 대여하고, 자유롭게 반납할 수 있는 **도시형 공유 우산 플랫폼**입니다.  
마스코트인 **우모(Umo)** 는 사용자에게 귀엽고 신뢰감 있는 가이드 역할을 하며, 비 오는 날 당신의 곁을 지켜줍니다.

### 핵심 기능
- 🔍 실시간 우산 위치 확인
- 📍 대여소 기반 우산 예약 / 반납
- 🌧️ 날씨 기반 스마트 알림
- 🧾 이용 내역 & 포인트 시스템
